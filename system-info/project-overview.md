# UniFlow Project Overview

## Project Name
UniFlow

## Project Type
Open-source student productivity platform

## Project Definition
UniFlow is an open-source web-based system designed to help university students manage schedules, to-do lists, assignments, and related files in one centralized platform.

## Main Modules
- Scheduling Module
- To-Do List Module
- Assignment Tracking Module
- File Organization Module
- Announcements and Reminder Module

## Current Development Status
Early Development / Phase 1

## Phase 1 Focus
Phase 1 focuses on:
- creating the project web pages,
- establishing the GitHub repository,
- preparing documentation,
- defining contribution guidelines,
- creating an open-source project structure.

## Future Vision
In future phases, UniFlow will evolve into a fully functional student support platform with improved usability, stronger collaboration features, and expanded productivity tools.
