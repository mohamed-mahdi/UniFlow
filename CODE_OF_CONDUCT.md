# Code of Conduct

## Our Commitment
We are committed to making participation in UniFlow a respectful and welcoming experience for everyone.

## Expected Behavior
- Be respectful and professional
- Accept constructive feedback
- Support collaboration
- Focus on what is best for the project and community

## Unacceptable Behavior
- Harassment or abusive language
- Personal attacks
- Discrimination
- Disruptive or unprofessional conduct

## Enforcement
Project maintainers may remove or reject contributions that violate this code of conduct.

## Scope
This code of conduct applies to all project spaces, including repository discussions, issue tracking, pull requests, and communication channels.
