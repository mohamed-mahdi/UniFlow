# UniFlow FAQ

## What is UniFlow?
UniFlow is an open-source productivity tool created to help university students manage schedules, tasks, assignments, and related files.

## Is UniFlow free?
Yes. UniFlow is a free and open-source software project.

## Who can use UniFlow?
Any university student who wants to organize academic and personal responsibilities.

## How can I report a bug?
Bugs can be reported through the GitHub Issues section.

## How can I contribute to the project?
You can contribute by forking the repository, making changes, and submitting a pull request according to the contribution guidelines.

## What features does UniFlow provide?
UniFlow provides scheduling, task tracking, assignment management, file organization, reminders, and project announcements.

## Is UniFlow complete?
No. UniFlow is currently in the early development stage as part of Phase 1.
