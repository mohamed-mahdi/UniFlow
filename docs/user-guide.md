# UniFlow User Guide

## Introduction
UniFlow is an open-source student productivity platform designed to help university students manage their academic and personal needs in one place.

## Main Functions
- Manage schedules and events
- Create and track to-do lists
- Monitor assignment deadlines
- Organize related study files
- View reminders and announcements

## How to Use UniFlow
1. Open the UniFlow website or application.
2. Navigate to your dashboard.
3. Add your tasks and assignments.
4. Set deadlines and schedules.
5. Attach or organize related files.
6. Track your progress regularly.

## Benefits for Students
UniFlow helps students improve time management, stay organized, and reduce missed deadlines by centralizing important academic tools in one system.

## Support
Users can report bugs and request features through the GitHub Issues page.
