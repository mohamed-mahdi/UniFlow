# Contributing to UniFlow

Thank you for your interest in contributing to UniFlow.

## How to Contribute
1. Fork the repository.
2. Clone your fork locally.
3. Create a new branch for your feature or fix.
4. Make your changes clearly and consistently.
5. Test your changes if applicable.
6. Commit your work with a meaningful commit message.
7. Push your branch and open a pull request.

## Contribution Rules
- Follow the project structure.
- Write clear and readable code.
- Keep documentation updated.
- Be respectful in discussions and reviews.
- Make small, focused contributions where possible.

## Reporting Issues
Use the GitHub Issues section to report bugs, request features, or suggest improvements.

## Communication
Contributors should use the official project channels such as GitHub Issues and GitHub Discussions.
