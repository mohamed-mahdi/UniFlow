About Contribution

UniFlow is an open-source project that welcomes developers, designers, and contributors who want to improve student productivity tools.

GitHub Repository

All project source code, documentation, and development tasks are managed through GitHub.

Visit UniFlow Repository

Participation Guidelines

  1. Fork the UniFlow repository.
  2. Create a new branch for your feature or fix.
  3. Follow project structure and coding standards.
  4. Test your changes before submission.
  5. Submit a pull request with a clear description.

Code of Conduct

All contributors must follow respectful and professional behavior when participating in the project.

  • Be respectful and collaborative
  • Accept constructive feedback
  • Avoid offensive or harmful behavior
  • Support a positive development environment

Communication & Message Forum

Contributors can communicate and collaborate using the following channels:

  • GitHub Issues (for bug reports and feature requests)
  • GitHub Discussions (for ideas and community interaction)
  • Pull Requests (for code contributions and reviews)

Development Information

The GitHub repository includes:

  • System-related documentation
  • Development tasks distribution
  • Source code and modules
  • Project progress updates
